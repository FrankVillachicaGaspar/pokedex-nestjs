# Pokedex API

## Stack

- MongoDB.
- NestJS.

## Ambiente de desarrollo

1. Clonar el repositorio.
2. Instalar Nest CLI instalado.

```shell
pnpm install @nestjs/cli
```

3. Instalar las dependencias.

```shell
pnpm install
```

4. Levantar la base de datos en docker

```shell
docker-compose up -d
```
